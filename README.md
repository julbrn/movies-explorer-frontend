## Доступные команды

### `npm start`

Открывает сайт в режиме разработки на локальном сервере [http://localhost:3000](http://localhost:3000).

### `npm run build`

Создаёт оптимизированную сборку приложения в папке `build`, когда сайт готов к развёртыванию в продакшене.
